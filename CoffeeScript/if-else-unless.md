###Switch/When/Else

```
switch day
	when 'mon' then go work 
	when 'tue' then go relax
	when 'fri','sat'
		if day is bingoDay
			go bingo
	else go work 
```

```
score = 76
grade = switch
	when score < 60 then 'D'
	when score < 70 then 'C'
	when score < 80 then 'B'
	else 'A'
```
